Display inline vs inline blocks

Aim:

  • Know the differences between inline vs inline-block
  • Make an example of inline and inline-block

Every HTML element has a default display value, depending on what type of element it is.
The two most common display values are block and inline.


Display: Inline

  • An inline element does not start on a new line
  • It only takes up as much width as necessary
  • Top and bottom margins/paddings are not respected

Display: Block

  • A block element always starts on a new line
  • Browsers add some space (a margin) before and after the element
  • Always takes up the full width available (stretches out to the left and right as far as it can)

e.g.


Display: inline-Block

  • Inline-block element allows you to set a width and height on the element
  • Top and bottom margins are respected
  • Does not add line-break after the element, so the element can sit next to other element
  • Commonly used to display list items horizontally instead of vertically
